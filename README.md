📝 Sobre o Projeto
O Estrelas do Amanhã é uma página de captura para matrícula de alunos em uma escola de educação infantil. O projeto foca em uma experiência de usuário (UX) fluida, 
utilizando um layout dividido onde as informações visuais permanecem fixas enquanto o usuário interage com o formulário.

Este projeto foi desenvolvido como parte do desafio prático de formulários e posicionamento avançado da Rocketseat.

------------------------------------------------------------------------------------------------------------------------------------------------------------------

🛠️ Tecnologias Utilizadas
O projeto foi construído utilizando as melhores práticas de desenvolvimento Web moderno:

HTML5: Estruturação semântica de formulários, inputs e labels.

CSS3:

Flexbox & Grid: Para o layout responsivo e divisão de colunas.

CSS Nesting: Uso da nova funcionalidade de aninhamento nativo do CSS.

Custom Properties (Variáveis): Para gestão de cores e tipografia.

Google Fonts: Utilização da fonte Poppins para uma leitura mais amigável.
